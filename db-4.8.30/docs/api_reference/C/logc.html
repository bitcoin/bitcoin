The DB_LOGC Handle 

#include <db.h>

typedef struct __typedef struct __db_log_cursor DB_LOGC;

The DB_LOGC object is the handle for a cursor into the log files, supporting sequential access to the records stored in log files. The handle is not free-threaded. Once the DB_LOGC->close() method is called, the handle may not be accessed again, regardless of that method's return.

For more information, see the DB_LSN handle.
