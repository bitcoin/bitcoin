DbEnv::get_msgfile()

#include <db_cxx.h>

void
DbEnv::get_msgfile(FILE **msgfilep);

The DbEnv::get_msgfile() method returns the FILE * used for displaying messages. This is set using the DbEnv::set_msgfile() method.

The DbEnv::get_msgfile() method may be called at any time during the life of the application.

Parameters

msgfilep

The DbEnv::get_msgfile() method returns the FILE * in msgfilep.

Class

DbEnv
