# Satoshi Vission (experimental)

Este directorio reúne tooling experimental para explorar ideas de operación y observabilidad del nodo, sin introducir cambios en reglas de consenso ni en validación de bloques. Su objetivo es servir como espacio de prueba y documentación técnica complementaria.

**Importante:** estas herramientas no modifican el comportamiento de consenso de Bitcoin Core. Cualquier propuesta aquí es un borrador y no debe interpretarse como especificación final.
